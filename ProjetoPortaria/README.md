# Projetos Python
---
## 🏢 **Sistema de Portaria**

Projeto desenvolvido em Python para gerenciamento de moradores e encomendas de condomínios.

O sistema permite cadastrar moradores, registrar encomendas, controlar retiradas e consultar informações de forma simples através do terminal.

**Funcionalidades:**
  * Cadastro de moradores
  * Cadastro de encomendas
  * Registro de retirada
  * Consulta de moradores
  * Consulta de encomendas
  * Persistência dos dados em JSON
  * Suporte a múltiplos edifícios

**Tecnologias:**

  * Python
  * JSON
  * Rich
  * Programação modular
